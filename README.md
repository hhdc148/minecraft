Minecraft 字体

📝 Minecraft Seven 字体

> 🔤 格式: OpenType (.otf)

🎨 用途: 设计、海报、Minecraft 风格相关项目

安装方法

Windows:

```
1. 右键点击字体文件
2. 选择"安装"
```

macOS:

```
1. 双击字体文件
2. 点击"安装字体"
```

Linux:

```bash
# 复制字体到系统字体目录
cp Minecraft-Seven-*.otf ~/.local/share/fonts/

# 刷新字体缓存
fc-cache -f -v
```

Android/iOS:
- 使用第三方字体管理器应用（如 iFont、FontFix 等）
